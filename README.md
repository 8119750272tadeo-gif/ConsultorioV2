# ConsultorioV2
Proyecto desarrollado en Java que permite la administracion basica de un consultorio medico, permite el alta de medicos, pacientes y citas. El proyecto utiliza Maven y esta confirgurado para generar FAT JAR.

## Instalación y configuración
Apache Netbeans 
Git 

## Uso del programa
usuario: tadeo
contraseña: 123
El sistema muestra un menu con las siguientes opciones: 
1. Dar de alta medico
2. Dar de alta paciente
3. Crear cita
4. Mostrar medicos
5. Mostrar pacientes
6. Mostrar citas
7. Salir
Los datos se almacenan en archivo de texto en formato CSV dentro de la carpeta db.
